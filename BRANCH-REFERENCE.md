## Career Paths by Branch

### CS - Computer Science
- **Early Career**: Junior Developer → Senior Developer (3-5 years)
- **Mid Career**: Tech Lead → Engineering Manager (5-8 years)
- **Senior Career**: Principal Engineer → Engineering Director (8+ years)
- **Average Salary Range**: $80K - $250K+ (varies by location/company)

### IT - Information Technology
- **Early Career**: Support Engineer → Systems Administrator (2-4 years)
- **Mid Career**: Infrastructure Engineer → IT Manager (4-7 years)
- **Senior Career**: Solutions Architect → IT Director (7+ years)
- **Average Salary Range**: $60K - $200K+ (varies by location/company)

### ENTC - Electronics & Telecommunication
- **Early Career**: Junior Engineer → Design Engineer (2-4 years)
- **Mid Career**: Senior Engineer → Project Lead (4-7 years)
- **Senior Career**: Principal Engineer → Engineering Manager (7+ years)
- **Average Salary Range**: $70K - $180K+ (varies by location/company)

### ECE - Electronics & Communication Engineering
- **Early Career**: Design Engineer → Hardware Developer (2-4 years)
- **Mid Career**: Senior Designer → Engineering Manager (4-7 years)
- **Senior Career**: Principal Engineer → Director of Engineering (7+ years)
- **Average Salary Range**: $75K - $200K+ (varies by location/company)

### AIDS - Artificial Intelligence & Data Science
- **Early Career**: Junior Data Scientist → ML Engineer (1-3 years)
- **Mid Career**: Senior ML Engineer → Research Scientist (3-6 years)
- **Senior Career**: Principal Scientist → ML Director (6+ years)
- **Average Salary Range**: $90K - $300K+ (varies by location/company)

## Common Mistakes in Data Entry

| Mistake | Example | Correction |
|---------|---------|------------|
| Branch Name Instead of Code | "Computer Science Engineering" | Use "CS" |
| Space in Branch Code | "C S" instead of "CS" | Remove spaces: "CS" |
| Lowercase Branch Code | "cs" instead of "CS" | Use uppercase: "CS" |
| Invalid Branch Code | "CSE" or "CSIT" | Use exact code: "CS" or "IT" |
| Missing Required Field | Missing Email | Ensure all required fields present |
| Invalid Email Format | "john@example" | Use proper email: "john@example.com" |
| Wrong Graduation Year | "2025" (future) | Use past year: 2024 or earlier |
| Special Characters in Name | "John@Doe" | Use normal names: "John Doe" |

## Troubleshooting Guide

### "Invalid branch" Error
- **Solution**: Use only CS, IT, ENTC, ECE, or AIDS
- **Check**: Spelling and capitalization

### "Invalid email format" Error
- **Solution**: Ensure email has format: name@domain.com
- **Example**: john.doe@company.com

### "Invalid graduation year" Error
- **Solution**: Use year between 1950 and current year + 5
- **Example**: 2020 (valid), 2025 (if current year is 2024)

### "Name is required" Error
- **Solution**: Ensure Name field is not empty
- **Check**: Row has actual name value

### "Email is required" Error
- **Solution**: Ensure Email field is not empty
- **Check**: Row has valid email address
